1. Pow(x,n) x^n (50 题),x=2,n=10, 2^10 = 1024
   输入 2.00000,10
   输出: 1024.00000
   输入 2.1000,3
   输出: 9.26100
   输入: 2.00000,-2
   输出: 0.25000
   方法 1: 调用库函数 O(1)
   方法 2: 暴力: O(N)
   方法 3: 分治,

```python
# 从中间分隔,判断是奇数还是偶数
# 1. 如果是偶数,只需要计算前一半的值,然后乘结果
# 2. 如果是奇数,拿掉一个x,然后在去算前一半的值,然后乘结果,在乘一个x
# 3. 分治的核心,再次拆分,每次减半,做递归
# O(logn)时间复杂度,每次减少一半
def myPow(self,x,n):
  if not n:
  return 1
  if n<0:
    return 1/self.myPow(x,-m)
  # 不能被2整除
  if n%2:
    return x * self.myPow(x,n-1)
  return self.myPow(x*x,n/2)

def myPow (self,x,n):
  if n <0:
    x = 1 /x
    n = -n
  pow = 1
  while n:
    if n & 1:
      pow *=x
      x *=x
      n>>1
  return pow
```

   输入[1,3,3,2,3]

   输入[1,1,1,0,2]
